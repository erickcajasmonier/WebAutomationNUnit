# WebAutomationNUnit

NuGet Packages used for this AUT project:
- Microsoft.NET.Test.Sdk
- NUnit
- NUnit3TestAdapter
- Selenium.Support
- Selenium.WebDriver
- DotNetSeleniumExtras.WaitHelpers (This package was used to manage waits using ExpectedConditions).
- Selenium.InternetExplorer.WebDriver
- Selenium.WebDriver.ChromeDriver
- Selenium.WebDriver.GeckoDriver

Supported browser versions:
- Google Chrome v84.0.4147.89
- Firefox v78.0.2
- Microsoft Edge v17.17134
- Internet Explorer v11.0

To run the test you only need to click on "Run All Tests" from the Test Explorer window.

To have in mind:
- The only class to edit data (just if you want) as you prefer is the class inside Tests folder, consider that you are able to change your name, last name, day of bith, month of birth, year of birth and state. This was made to let you test with your own information.
